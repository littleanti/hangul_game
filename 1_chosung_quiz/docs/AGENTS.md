## For AI Agents

### Working In This Directory
- 코드 변경과 무관하게 문서만 업데이트하는 것은 자유롭게 가능.
- 수동 테스트 체크리스트는 `TRD.md` §8을 기준으로 한다.
- 자동화 테스트 런너 없음 — 새 기능 추가 시 TRD에 체크리스트 항목도 추가 권장.
- **시리즈 공통 UI 기준**: `TRD.md §10 홈·설정·완료 화면 디자인 시스템`이 2 ~ 6단계 게임의 디자인 레퍼런스. 이 섹션을 수정하면 시리즈 전체 설계 기준이 바뀌므로 주의.
- `PLAN.md ##디자인 기준` 체크리스트는 2 ~ 6단계가 준수해야 할 구현 항목의 출발점 스펙이다.
