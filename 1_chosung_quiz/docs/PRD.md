# 📋 PRD — 초성 퀴즈 v2

> Product Requirements Document
> Last updated: 2026-05-05

## 1. 제품 개요

**제품명**: 초성 퀴즈 — 그림 보고 단어 맞히기

**한 줄 요약**: 그림(이모지 또는 이미지)을 보고 한글 초성 힌트로 단어를 맞히는 웹 기반 학습 게임.

**배포 환경**: 단일 페이지 정적 웹앱 (모바일 웹 포함)

## 2. 타겟 사용자

| 사용자 그룹 | 특징 | 주요 니즈 |
|---|---|---|
| 한글 배우는 아이 (5-8세) | 글자를 이제 막 읽기 시작 | 쉬운 단어, 그림, 발음 듣기 |
| 한국어 학습자 (외국인) | 한국어 기초-중급 | 카테고리별 어휘 확장 |
| 일반 사용자 | 두뇌 게임 목적 | 빠른 진행, 난이도 조절 |
| 부모/교사 | 아이 교육용으로 사용 | 설정 커스터마이징 |

## 3. 사용자 시나리오

### S1. 아이가 혼자 놀기
- 유치원생이 태블릿으로 열고 "바로 시작" 터치
- 이모지를 보고 초성을 맞혀봄
- 모르면 🔊 버튼으로 발음을 듣고 추측

### S2. 부모가 세팅해주기
- 설정 화면에서 "동물"만 선택, 5문제, 쉬움, 타이머 끄기
- 아이에게 넘겨줌
- 종료 후 틀린 문제 복습

### S3. 한국어 학습자 셀프 학습
- "음식" 카테고리, 20문제, 10초 타이머, 어려움
- TTS로 발음 확인
- 틀린 단어는 복습 리스트에서 반복 재생

## 4. 기능 요구사항

### P0 (필수)

|---|---|---|

### 추가 구현 (v2.2, 완료)
| ID | 기능 | 설명 |
|---|---|---|
| F19 | 글자 선택 모드 (`inputMode`) | 초성 힌트 대신 음절 버튼을 탭하여 순서대로 단어를 입력. 방해 음절 8개 포함 풀에서 선택. 맞으면 자동 정답 처리, 틀리면 오답 처리 |
| F20 | 힌트 시스템 (`hintEnabled`) | 게임 중 힌트 버튼 표시/숨김 토글. 일반 모드: 초성→글자 점진적 공개. 글자 선택 모드: 다음 음절 자동 선택 |
| F21 | 레벨 버튼 (1~4) | 홈 화면에서 레벨(1=쉬움/2=보통/3=어려움/4=전체) 탭 시 즉시 게임 시작. 다른 설정은 유지 |
| F22 | 연속 플레이 중복 제한 | `lastGameWords` 추적으로 직전 게임 단어 재출제 비율 ≤ 20% 보장 (풀이 충분할 때) |

### 추가 구현 (v2.3, 완료)
| ID | 기능 | 설명 |
|---|---|---|
| F23 | 가로 모드 2컬럼 레이아웃 | 가로 모드(orientation: landscape) + 낮은 높이(max-height: 700px) 환경에서 플레이 화면을 자동으로 2컬럼 그리드로 전환. 왼쪽: 그림 카드 / 오른쪽: 글자 입력 영역 + 정답 확인 버튼. 모바일 가로/iPad 가로/저해상도 모니터 모두 자동 적응 |
| F24 | viewport 비례 컨텐츠 | 가로 모드에서 이모지·이미지·단어 글자·버튼 사이즈가 `clamp()` + vh/vw로 viewport에 비례 (작은 화면에서 자동 축소, 큰 화면에서 확장). 카드는 row 높이만큼 stretch되어 빈 공백 없이 꽉 채움 |

### P2 (향후)

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|---|---|
| 성능 | 문제 전환 지연 < 500ms, 60fps 애니메이션 |
| 호환성 | Chrome, Safari, Edge 최신 버전 / iOS Safari 15+ |
| 반응형 | 320px ~ 768px 모바일 우선, 데스크톱도 자연스럽게 |
| 접근성 | 터치 타겟 최소 44×44px, 색상 대비 WCAG AA |
| 오프라인 | 초기 로드 후 네트워크 없이도 작동 (폰트 제외) |
| 무장애 | TTS 미지원 시 자동 비활성화 등 graceful degradation |

## 6. 성공 지표

- **학습 효과**: 세션당 평균 5문제 이상 완료
- **재사용**: 한 번 이상 "다시 하기" 누르는 비율 40%+
- **설정 활용**: 설정 화면 진입율 30%+ (의도적 학습)
- **오답 복습**: 종료 화면에서 복습 리스트 펼침 50%+

## 7. 범위 제한 (Out of Scope)

- 회원가입/로그인 — 없음, 모든 것은 로컬
- 데이터 수집 — 외부 서버로 전송하지 않음
- 멀티플레이어
- 백엔드 / 데이터베이스

## 9. 홈·설정·완료 화면 디자인 규격 (시리즈 공통 기준)

본 게임(`1_chosung_quiz`)의 홈 화면·설정 화면·게임 완료 화면 디자인은 **7단계 시리즈 전체의 UI 디자인 기준(Design Reference)**이다. 2~6단계 게임의 시작·설정·완료 화면은 아래 규격을 따른다.

| 디자인 요소 | 규격 |
|---|---|
| 제목 폰트 | **Jua** (Google Fonts) |
| 설명·본문 폰트 | **Gowun Dodum** (Google Fonts) |
| 버튼 디자인 | 이 게임의 `src/css/components.css` 버튼 스타일 — 큰 라운드 버튼, CSS 변수 색상 |
| 색감 | `src/css/tokens.css` CSS 변수 팔레트 |
| 배경 | `--color-bg` 등 `tokens.css` 공통 토큰 |

> 플레이 화면 등은 각 게임 특성에 맞게 확장 가능하다.  
> 단, 시작·설정·완료 화면은 이 게임의 디자인 언어를 기준으로 통일한다.
