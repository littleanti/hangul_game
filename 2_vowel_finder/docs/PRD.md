# PRD — 모음 찾기 (2_vowel_finder)

> Product Requirements Document
> Last updated: 2026-06-10

---

## 1. 제품 개요

**제품명**: 모음 찾기 — 소리로 모음 재인하기

**한 줄 요약**: 단일 모음 글자와 TTS 소리를 듣고 같은 소리·형태의 모음을 탭으로 매칭하며, 세로형/가로형 두 통으로 분류하고 짧은 드래그 온보딩을 경험하는 유아용 웹 학습 게임.

**배포 환경**: 단일 페이지 정적 웹앱 (모바일/태블릿 우선, PWA)

**개발 포트**: 4328

**시리즈 위치**: Stage 2 — 음절 조립소(2_syllable_assembly)의 Level 0 In-stage 온보딩 모듈

## 2. 문제 정의 및 학습 갭 근거

> 본 섹션은 `LEARNING_PROGRESSION_ANALYSIS.md` §5.3 및 §3.1의 분석을 인용한다.

### 2.1 메우는 갭 (D5 — 선행 기술 공백)

Stage 1(초성 퀴즈)은 자음 19자의 초성 인출만 자동화하며, 한글 음절 조립에 필수적인 **중성(모음) 재인**을 학습 목표로 삼지 않는다. Stage 2(음절 조립소)는 첫 라운드부터 다음을 전제한다.

| Stage 2 전제 조건 | Stage 1 출구 여부 |
|---|---|
| 10개 기본 모음 자소-음소 연결 재인 | **부재** — S1은 초성(자음) 전용 |
| 세로형 모음(ㅏ ㅓ ㅗ ㅜ ㅡ) vs 가로형 모음(ㅑ ㅕ ㅛ ㅠ ㅣ) 범주 구별 | **부재** |
| ±20dp 드래그 운동 협응 | **부재** — S1은 탭 전용 |

`§3.1 D5 = 4`로 채점된 이 공백은 "2~3개 미보강 선행 기술"로서, 브릿지 없이 S2에 직행하면 모음 재인·형태 범주·드래그 운동이 모두 신규로 쏟아진다. `§5.3`의 처방: **모음 인지 + tap→drag 운동 온보딩을 Stage 2 Level 0 In-stage 모듈로 내장**.

### 2.2 완충하지 않는 갭 (비목표 — §5.3 명시)

`§3.1 D4 = 4`의 연령 역행(Stage 1: 5~8세 → Stage 2: 만 4~6세)은 로드맵 순서 결함이며, 어떤 브릿지로도 고칠 수 없다. 본 게임은 이 D4 문제를 해결하려 하지 않는다.

## 3. 대상 사용자 및 발달 정합

| 사용자 그룹 | 특징 | 주요 니즈 |
|---|---|---|
| 만 4~6세 유아 (핵심 코호트) | 전조작기 후기~구체적 조작기 진입, 자모 낱자 재인 시작, 소근육 정밀도 낮음 | 큰 타겟(64dp+), TTS 주도, 즉각 긍정 피드백 |
| Stage 2(음절 조립소) 직진입 아동 | Stage 1 미경험 가능 — 본 게임이 첫 한글 게임일 수도 있음 | 모음 글자와 소리의 첫 만남이 부담 없을 것 |
| 부모/교사 | 아이 화면 방치 불안 | 짧은 세션(3~5분), 안전한 비 IME 입력 |

**발달 정합 근거**

- **Vygotsky ZPD**: 탭 매칭(수용 재인)은 독립 수행 가능 수준, 짧은 드래그(운동 온보딩)는 조력 하 수행 — 두 단계가 ZPD 상단에 정확히 위치.
- **Ehri 철자 발달**: partial-alphabetic 단계 초입 — 낱자(자소)-소리(음소) 연결을 형성하는 시기. 모음 재인이 이 연결의 절반을 구성.
- **피아제 전조작기**: 구체적 물체(글자 카드)를 조작하고 범주에 담는 활동(세로형/가로형 분류)이 적합. 추상 규칙 암기보다 물리적 분류·조작 우선.
- **소근육 발달**: 만 4~5세는 목표점까지의 정밀 드래그가 아직 어려움 — ±20dp 자성 스냅이 필수 비계.

## 4. 학습 목표

### 4.1 획득 기술 (본 게임 출구에서 달성)

| # | 획득 기술 | 평가 방식 |
|---|---|---|
| G1 | 10개 기본 모음(ㅏ ㅑ ㅓ ㅕ ㅗ ㅛ ㅜ ㅠ ㅡ ㅣ) 자소-음소 재인 자동화 | 탭 매칭 정확도 ≥ 80% |
| G2 | 세로형 모음(ㅏ ㅑ ㅓ ㅕ ㅗ ㅛ ㅜ ㅠ ㅡ) vs 가로형 모음(ㅣ) + 형태 범주 직관 | 분류 드래그 성공률 ≥ 75% |
| G3 | ±20dp 드래그 운동 첫 경험 — 짧은 이동 + 자성 스냅 체험 | 드래그 온보딩 1스텝 완료 |

> 보다 엄밀한 분류: 한국어 모음을 현대 문법에서는 단모음 10개(ㅏ ㅐ ㅓ ㅔ ㅗ ㅚ ㅜ ㅓ ㅡ ㅣ)로 보지만, 본 게임은 유아 교육 관행상 **형태·위치 기준 세로형/가로형** 으로 분류하여 Stage 2의 자모 배치 도크(세로/가로 구분)와 직결시킨다.

### 4.2 비목표 — 인접 단계로 명시적 유보

| 유보 항목 | 유보 이유 | 담당 단계 |

| 초성(자음) 19자 재인 및 인출 | Stage 1 출구 기술 — 중복 교육 금지 | Stage 1 |
| 받침(종성) 재인 | Stage 2 메인 콘텐츠 (CV+받침 조합) | Stage 2 |

| 한글 IME 타이핑 | 전 시리즈 IME 회피 원칙 (AGENTS.md §3) | 없음 / Stage 7+ |

## 7. 콘텐츠 / 데이터 범위

### 7.1 대상 모음 목록 (10개 기본 모음)

| 분류 | 모음 | TTS 발음 예시 | 형태 특징 |
|---|---|---|---|
| 세로형 | ㅏ ㅑ ㅓ ㅕ | "아 야 어 여" | 세로 기둥 + 오른쪽/왼쪽 가로획 |
| 세로형 | ㅗ ㅛ ㅜ ㅠ | "오 요 우 유" | 가로 기둥 + 위/아래 세로획 |
| 가로형 | ㅡ | "으" | 가로선 단독 |
| 기타 단형 | ㅣ | "이" | 세로선 단독 |

> 이중 모음(ㅐ ㅔ ㅘ 등)은 본 게임 범위 밖 — Stage 2에서 복합 음절 조립 시 자연스럽게 노출.

### 7.2 Level 0 문항 구성 (소리 매칭)

| 라운드 | 정답 모음 | 보기 구성 | 오답 유형 |
|---|---|---|---|
| 1 | ㅏ | ㅏ ㅓ ㅗ ㅜ | 발음 유사 오답 |
| 2 | ㅣ | ㅣ ㅡ ㅏ ㅓ | 형태 유사 오답 |
| 3 | ㅗ | ㅗ ㅛ ㅜ ㅠ | 이중/파생 혼동 오답 |
| 4 | ㅑ | ㅏ ㅑ ㅕ ㅓ | 기본형 vs 이중형 |
| 5 | ㅡ | ㅡ ㅣ ㅗ ㅜ | 형태 유사 오답 |

> 5라운드는 예시이며, 10개 모음 전체를 커버하도록 랜덤 출제. 같은 라운드에서 정답이 반복되지 않도록 풀 관리.

### 7.3 Level 1 문항 구성 (형태 분류)

- 10개 기본 모음을 1회 순환 — 순서는 셔플
- 각 문항: 모음 카드 1장 제시 → 세로형 통 / 가로형 통 중 하나에 배정
- 세로형 9개(ㅏ ㅑ ㅓ ㅕ ㅗ ㅛ ㅜ ㅠ ㅡ) / 가로형 1개(ㅣ) — 실제 분류 기준은 아동 교육 자료 기준 적용
- 3단 비계: L1 통 이름 레이블 + 예시 모음 표시 → L2 통 이름 레이블만 → L3 빈 통 (아이콘만)

### 7.4 드래그 온보딩 스텝

- 화면: 모음 카드 1장 (ㅏ 고정) + 목표 영역(네모 칸)
- 거리: 화면 폭의 약 20% 이내 이동 (±20dp 스냅 반경)
- 가이드: 점선 화살표 + "이쪽으로 끌어봐요!" 텍스트 + TTS
- 성공 기준: 목표 영역 ±20dp 내에 드롭 → 자성 흡착 + 효과음 + 완료 애니메이션

## 8. 성공 지표

| 지표 | 목표값 | 측정 방법 |
|---|---|---|

## 9. 시리즈 연속성 요구

### 9.1 이전 단계 — Stage 1 (초성 퀴즈, `1_chosung_quiz`)

| 연결 항목 | 요구 사항 |
|---|---|
| 진입 경로 | Stage 1 완료 후 "다음 단계" 버튼 또는 Stage 2 홈에서 독립 진입(Stage 1 미경험 아동 허용) |
| 선행 전제 데이터 | Stage 1 출구 기술(초성 인출)은 본 게임에서 사용하지 않음 — 모음 전용 콘텐츠 |

| 화면 모드 | Stage 1: 세로 → 본 게임: **세로 유지** (Stage 2의 가로 전환 전 마지막 세로 단계) |

### 9.2 다음 단계 — Stage 2 (음절 조립소, `2_syllable_assembly`)

| 연결 항목 | 요구 사항 |
|---|---|
| 출구 기술 전달 | G1(모음 10개 재인) + G2(세로/가로 형태 범주) + G3(±20dp 드래그 체험) — 3가지 모두 Stage 2 Level 1의 선행 |
| 드래그 연속성 | 본 게임 ±20dp 스냅 맛보기 → Stage 2의 자모 슬롯 드래그(±20dp 자성 스냅 동일 반경) 으로 운동 습관 이어짐 |

| 데이터 호환 | localStorage 키 접두사: `vowel_finder_` (Stage 2: `syllable_assembly_`) — 네임스페이스 충돌 없음 |

| 항목 | 규격 |
|---|---|
| Service Worker CACHE_VERSION | `vowel-finder-v1` (게임별 고유 접두사) |
| manifest start_url / scope | `'./'` (상대 경로) |
| localStorage 접두사 | `vowel_finder_` |
| TTS | Web Speech API, `ko-KR` 우선, 미지원 시 graceful degradation (토글 자동 비활성화) |
| 화면 회전 잠금 | 세로 권장 — Stage 1과 동일 |

## 10. 범위 제한 (Out of Scope)

- 회원가입/로그인 — 없음, 모든 데이터는 로컬
- 외부 서버 전송 — 없음
- 이중 모음(ㅐ ㅔ ㅘ ㅙ ㅚ ㅝ ㅞ ㅟ ㅢ) 학습 — Stage 2 이상 배정
- 자음 학습 — Stage 1 담당
- 음절 조립 — Stage 2 담당
- 멀티플레이어
- 백엔드 / 외부 데이터베이스
