# 🎯 사자성어 크로스워드

사자성어(四字成語)로 즐기는 한국어 크로스워드 퍼즐 게임입니다.  
순수 HTML / CSS / JavaScript로 구현되어 별도 설치 없이 브라우저에서 바로 플레이할 수 있습니다.

## 플레이 방법

`index.html`을 브라우저로 열면 바로 시작됩니다.

```
index.html  ←  더블클릭 또는 브라우저로 열기
```

## 기능

- **자동 배치** — 매 판마다 사자성어 6개를 무작위로 골라 크로스워드를 자동 생성
- **힌트** — 현재 단어의 미입력 칸 하나를 공개 (점수 차감)
- **정답 보기** — 전체 정답 공개
- **채점** — 입력한 답을 즉시 확인하고 점수 계산
- **새 게임** — 언제든지 새 판 시작
- **한국어 IME 완벽 지원** — 조합 중인 한글을 실시간으로 미리보기
- **키보드 탐색** — 방향키, Tab, Backspace, Enter 지원
- **모바일 대응** — 터치 입력 및 소프트 키보드 지원

## 조작법

| 입력 | 동작 |
|------|------|
| 셀 클릭 | 해당 칸 선택 (같은 셀 재클릭 시 방향 전환) |
| 단어 목록 클릭 | 해당 단어로 이동 |
| 방향키 | 인접 셀로 이동 |
| Tab / Shift+Tab | 다음 / 이전 단어로 이동 |
| Backspace | 현재 칸 삭제 → 빈 칸이면 이전 칸으로 이동 |
| Enter | 채점 |

## 파일 구조

```
├── index.html     # 마크업 및 레이아웃
├── style.css      # 스타일
├── data.js        # 사자성어 데이터 (50개+)
├── crossword.js   # 크로스워드 배치 알고리즘
└── app.js         # 게임 로직 및 IME 처리
```

## 기술 스택

- Vanilla HTML / CSS / JavaScript (의존성 없음)
- CSS Grid 기반 크로스워드 렌더링
- Hidden-input 아키텍처로 한국어 IME 조합 처리
